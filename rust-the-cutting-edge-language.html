Rust the Cutting Edge Language!

Date 6/4/2015 Author Kori  | Category : Computers

 

So yesterday I started rust, because a few weeks/months ago my dad told me to pick out a brand new language that was barely out of beta this year. The reason why I picked Rust over Go and Scala was because I thought it was the least Java and C like. After trying to learn it for an hour with my dad, I learned the sad truth that it's like a mix of C and Java, my most hated languanges. So if you like those two languages a bunch totally, try Rust I bet you'll love it but if your like me don't pick it. If you don't however, I'd suggest choosing none of them because Scala is produced to help Java and Go looks like C but has background intelligence.
