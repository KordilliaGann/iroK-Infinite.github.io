New Coder.io

Date 3/22/2015 Author Kori  | Category : Computers

 

Last week during math intensive Professor Welch (who is one of the leaders of the CSIT-In-3) suggested that I start doing http://newcoder.io/, so I talked to my dad about it and we added it to my list of daily programming. Setting up stuff is sometimes difficult for me, this time I was trying to download stuff using the Windows way on my Linux boot (my laptop is dual-booted with Windows 8.1 and Ubuntu 14.04). I was wondering what was wrong, I even asked my dad to help which is when he noticed what I had been trying to do. I also had problems downloading matplotlib, in a different way, but I managed to solve that myself with lots of help from captain internet. Now that I have actually managed to do the coding part, I discovered I kind of like it, because the tutorials are pretty nice and somewhat easy to do. Keep in mind I've only been able to work on it for two hours so far.
